# Syntecxhub_Personal_Voice_Assistant

A smart desktop Voice Assistant built using Python that can understand voice commands, respond with speech, and perform useful everyday tasks through a simple graphical interface.

The project combines Speech Recognition, Text-to-Speech (TTS), and GUI development to create an interactive assistant capable of opening websites, launching applications, and providing real-time information such as the current date and time.

---

## ✨ Key Features

### 🌐 Web Navigation

* Open YouTube instantly
* Open Google Search

### 🖥️ System Controls

## 📋 Supported Voice Commands

| Command              | Action                 |
| -------------------- | ---------------------- |
| Open YouTube         | Opens YouTube          |
| Open Google          | Opens Google           |
| Open GitHub          | Opens GitHub           |
| Open Calculator      | Launches Calculator    |
| Open Notepad         | Launches Notepad       |
| Open Paint           | Launches Paint         |
| What is the time     | Announces current time |
| What is today's date | Announces current date |
| Hello                | Greets the user        |
| Exit                 | Closes the assistant   |

---

## ⚙️ How It Works

1. The user clicks the **Start Listening** button.
2. The microphone captures the user's voice command.
3. Speech Recognition converts speech into text.
4. The assistant processes the command.
5. The corresponding task is executed.
6. The assistant provides both voice and on-screen feedback.
